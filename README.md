# 📜 Silent Farewell of Words (Kelimelerin Sessiz Vedası)

**Silent Farewell of Words** is an elegant, web-based digital flashcard and quiz application. It is designed to preserve and explore words from the Turkish language that are either fading from daily use, changing in meaning, or carry deep etymological roots. 

The application features a unique "Ancient Parchment" aesthetic, combining historical atmosphere with modern functionality.

## ✨ Key Features

* **Antique Aesthetic:** A meticulously designed interface featuring burnt-edge parchment, quill-and-ink motifs, and classic typography.
* **Rich Vocabulary Database:** Includes over 80+ curated words with their etymological origins (Arabic, Persian, Old Turkish, etc.).
* **Dual Meaning Insights:** Displays both modern definitions and historical "old meanings" where applicable.
* **Literary Context:** Provides authentic usage examples from renowned authors like Halide Edip Adıvar, Yahya Kemal Beyatlı, and Sait Faik Abasıyanık.
* **Interactive Quiz Mode:** A built-in gamified testing system to help users memorize and learn words effectively.
* **Export to Image:** Users can save their favorite word cards as PNG images using the integrated `html2canvas` library.
* **Live Search:** High-performance search bar for instant word filtering.

## 🚀 Technical Stack

This is a lightweight, serverless "Single Page Application" (SPA):
* **Frontend:** HTML5, CSS3 (Custom Border-Images & Google Fonts).
* **Logic:** Vanilla JavaScript.
* **Libraries:** [html2canvas](https://html2canvas.hertzen.com/) (for image rendering).
